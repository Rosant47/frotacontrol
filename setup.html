Configuração inicial do sistema

Pré-requisitos:
  1. Editar assets/js/config.js com as credenciais do Firebase
  2. Ativar Authentication → E-mail/Senha no Firebase Console
  3. Criar o banco Firestore Database (modo produção)
  4. Publicar as regras de segurança do arquivo firestore.rules
